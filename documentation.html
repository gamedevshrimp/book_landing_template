The basic structure of a project - top

All page content is placed in a single file, index.html.

Open it and find the main HTML tag. Inside this tag, you will find the main sections of the page, such as:

  • Product
  • Books
  • Testimonials
  • Gallery
  • Contact

Feel free to adjust the order of these sections if needed. To reorder the sections, simply copy the entire code of a section and paste it where you want it to appear. 


    <main>
        <!-- Product Section -->
        <section class="product-section">
        </section>
        
        <!-- Books Section -->
        <section class="book-section" id="portfolio">
        </section>
        
        <!-- Testimonials Section-->
        <section class="testimonials-section" id="testimonials">
        </section>
        
        <!-- Gallery Section-->
        <section class="gallery-section" id="gallery">
        </section>
        
        <!-- Contact Section-->
        <section class="contact-section" id="contact">
        </section>
        <!-- Footer Section-->
        <footer class="footer-section">
        </footer>
    </main>

Color and Font changing - top

All color and font settings are placed at the top of the style.css file. You can go there and check the :root tag for all the customizable variables. 


    :root {
    /* Colors */
    --white-color: #fff;
    --dark-color: #1d3557;
    --primary-color: #f1faee;
    --secondary-color: #e63946;
    --light-pink-color: #f5f5f7;
    --medium-gray-color: #457b9d;
    
    /* Font Size */
    --font-size-s: 0.9rem;
    --font-size-n: 1rem;
    --font-size-m: 1.12rem;
    --font-size-l: 1.5rem;
    --font-size-xl: 2rem;
    --font-size-xxl: 3rem;
    
    /* Font Weight */
    --font-weight-normal: 400;
    --font-weight-medium: 500;
    --font-weight-semibold: 600;
    --font-weight-bold: 700;
    
    /* Border Radius */
    --border-radius-s: 8px;
    --border-radius-m: 30px;
    --border-radius-circle: 50%;
    
    /* Site Max Width */
    --site-max-width: 1300px;
    }

Product section editing - top

The Product section is the first section that users will see on your site. To replace the product image, upload your picture to the ./images folder.

Then, edit the image path inside the div with the product-image-wrapper class attribute.

To change the text in this section, locate the div with the product-details class.

Here, you will find the title, subtitle, and description blocks. Feel free to update them as needed. 


    <section class="product-section">
        <div class="section-content">
            <div class="product-details">
                <h2 class="title">LitPage</h2>
                <h3 class="subtitle">Launch your book with a polished web solution.
                </h3>

                <img src="./images/book_product_photo.png" alt="Hero" class="product-image">
            </div>
        </div>
    </section>

About Section Editing - top

Then, find the <div class="about-image-wrapper"> and update the image path with your photo.

Books Section Editing - top

The Books section is where you can add links to your other books. First, upload the covers of your books to the ./images folder.

Then, locate the <li class="book-item"> tag and update the image path.

You can also modify the text of the description inside the same li tag. 

 
    <li class="book-item"> 
        <img src="./images/book_17.png" alt="" class="menu-image"> 
        <h3 class="name">Another Book Item</h3> 
        <p class="text"> Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. 
        </p> 
    </li>

Testimonials Section Editing - top

The Testimonials section is where you can showcase reviews or feedback from readers. To add new testimonial images, upload them to the `./images` folder.

Then, locate the `<li class="testimonial-item">` tag and update the image path.

You can also edit the testimonial text within the same `li` tag. 


    <li class="book-item">
        <img src="./images/book_17.png" alt="" class="menu-image">
        <h3 class="name">Another Book Item</h3>
        <p class="text">
            Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla
            pariatur.
        </p>
    </li>

Gallery Section Editing - top

The Gallery section does not contain editable text. However, you can always add your images to the ./images folder.

Then, update the image path in the <li class="gallery-item"> tag.

You can also edit the image captions (if any) within the same li tag. 


    <li class="gallery-item">
        <img src="./images/photo_1.jpeg" alt="#" class="gallery-image">
    </li>
